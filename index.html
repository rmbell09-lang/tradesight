✨ Features

🧬 AI Strategy Tournaments

Automated overnight evolution — the best strategy wins, the rest are retired. Runs while you sleep.

📊 15+ Technical Indicators

MACD, RSI, Bollinger Bands, EMA crossovers, ATR, volume analysis, and more. All built-in.

💸 Paper Trading

Connect Alpaca paper account — trade with fake money, track real P&L. Risk-free strategy validation.

📈 Backtest Engine

Test any strategy against historical data before risking a single dollar.

🌐 Web Dashboard

Local Flask dashboard with real-time P&L, active positions, and strategy leaderboards.

🔒 Self-Hosted

Quick Start

$ git clone https://github.com/rmbell09-lang/tradesight
$ cd tradesight
$ pip install -r requirements.txt
$ cp .env.example .env  # add your Alpaca key
$ python web/dashboard.py

Runs on Python 3.11+. No Docker required. Overnight AI tournaments run automatically.
